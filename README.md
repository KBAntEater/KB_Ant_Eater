# 개미집
## 💰 소개
소비 데이터를 기반한 배당주를 추천해주고, 금융에 특화된 모델을 활용하여 양질의 증권 기사 요약본을 제공하는 플랫폼
## 🛠 기술스택
- Django
- javascript
- scikit-learn
- KB-Albert(Tokenizer)
## 📌 주요 기능 요약
- 실시간 주가 트리맵
- 소비패턴항목에 따른 배당성장주 추천과 주가 예측
- 내가 보유한 주식 종목 관리
- 증권 기사 추출 요약 제공
